Glosario de Desarrollo Web

HTML (HyperText Markup Language)

Lenguaje de marcado para crear y estructurar páginas web.

CSS (Cascading Style Sheets)

Lenguaje de estilo para diseñar y personalizar la apariencia de las páginas web.

JavaScript

Lenguaje de programación que permite agregar interactividad a las páginas web.

DOM (Document Object Model)

Modelo que representa la estructura de un documento HTML o XML como una jerarquía de nodos.

AJAX (Asynchronous JavaScript and XML)

Técnica para actualizar partes de una página web sin recargar toda la página.

API (Application Programming Interface)

Conjunto de reglas que permite a aplicaciones interactuar entre sí.

Backend

Parte del desarrollo web que se encarga del procesamiento del servidor, bases de datos y lógica de la aplicación.

Frontend

Parte del desarrollo web que se ocupa de la interfaz de usuario y la experiencia del usuario.

Framework

Conjunto de herramientas y bibliotecas que facilitan el desarrollo de aplicaciones web.

Library

Colección de funciones y procedimientos reutilizables para realizar tareas comunes.

HTTP (Hypertext Transfer Protocol)

Protocolo utilizado para la transferencia de datos en la web.

HTTPS (Hypertext Transfer Protocol Secure)

Versión segura del HTTP que utiliza cifrado para proteger la transferencia de datos.

URL (Uniform Resource Locator)

Dirección que localiza un recurso en la web.

Web Server

Software que entrega contenido web a los navegadores.

Browser (Navegador)

Programa que permite a los usuarios acceder y visualizar sitios web.

Responsive Design

Diseño web que se adapta a diferentes tamaños de pantalla y dispositivos.

SEO (Search Engine Optimization)

Conjunto de prácticas para mejorar la visibilidad de un sitio web en los motores de búsqueda.

Cookie

Pequeño archivo de datos que un sitio web almacena en el navegador del usuario.

Session

Periodo de tiempo durante el cual un usuario interactúa con una aplicación web.

Database (Base de datos)

Sistema que almacena y organiza datos.

SQL (Structured Query Language)

Lenguaje de programación para gestionar y manipular bases de datos relacionales.

NoSQL

Tipo de base de datos que no utiliza el esquema de tablas tradicional de las bases de datos relacionales.

CMS (Content Management System)

Plataforma que permite a los usuarios crear, gestionar y modificar contenido digital sin conocimientos técnicos.

Web Hosting

Servicio que permite a los sitios web ser accesibles en Internet.

Domain Name

Nombre único que identifica a un sitio web en Internet.

SSL (Secure Sockets Layer)

Protocolo de seguridad para establecer una conexión cifrada entre un servidor y un cliente.

JSON (JavaScript Object Notation)

Formato ligero de intercambio de datos.

REST (Representational State Transfer)

Estilo arquitectónico para diseñar servicios web.

GraphQL

Lenguaje de consulta para APIs que permite solicitar solo los datos necesarios.

Version Control

Sistema para gestionar cambios en el código fuente de un proyecto.

Git

Sistema de control de versiones distribuido.

Repository (Repositorio)

Lugar donde se almacenan los archivos de un proyecto y su historial de versiones.

Deployment

Proceso de publicar una aplicación web en un servidor para que esté accesible en Internet.

CI/CD (Continuous Integration/Continuous Deployment)

Práctica de integrar y desplegar cambios en el código de manera continua.

Virtual DOM

Representación en memoria del DOM real que permite actualizaciones eficientes.

Component

Parte reutilizable y autónoma de una interfaz de usuario.

Single Page Application (SPA)

Aplicación web que se carga una sola vez y actualiza el contenido dinámicamente.

Progressive Web App (PWA)

Aplicación web que utiliza capacidades modernas para ofrecer una experiencia similar a una app nativa.

Webpack

Herramienta para empaquetar módulos de JavaScript y otros activos.

Babel

Compilador de JavaScript que permite usar características de JavaScript moderno en navegadores antiguos.

TypeScript

Superset de JavaScript que agrega tipos estáticos.

Node.js

Entorno de ejecución de JavaScript del lado del servidor.

Express.js

Framework de Node.js para construir aplicaciones web y APIs.

React

Biblioteca de JavaScript para construir interfaces de usuario.

Angular

Framework de desarrollo web para construir aplicaciones de una sola página.

Vue.js

Framework progresivo para construir interfaces de usuario.

Sass (Syntactically Awesome Stylesheets)

Preprocesador CSS que permite usar variables, anidamiento y más.

LESS (Leaner Style Sheets)

Preprocesador CSS similar a Sass.

WebSocket

Protocolo para comunicación bidireccional en tiempo real entre un cliente y un servidor.

MVC (Model-View-Controller)

Patrón de diseño que separa la aplicación en tres componentes: modelo, vista y controlador.
